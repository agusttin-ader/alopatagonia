# Alo Patagonia — sitio web

Sitio público de Alo Patagonia: asesoramiento y organización de viajes en la Patagonia.

## Requisitos

- Node.js compatible con la versión indicada por Next.js del proyecto

## Desarrollo

```bash
npm install
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000).

## Producción

```bash
npm run build
npm start
```

## Variables de entorno

Copiá `.env.example` a `.env.local` y completá los valores necesarios (por ejemplo WhatsApp).

## Notas técnicas

Ver `DEVELOPING.md` y `PROJECT.md` para convenciones del repo.
